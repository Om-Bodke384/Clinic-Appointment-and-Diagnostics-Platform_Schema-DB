# Hospital Management System - DB Schema

## Tables
- **Patient** - stores patient info
- **Doctor** - stores doctor info
- **Appointments** - patient books appointment with doctor
- **Consultations** - doctor gives medicine after appointment
- **Medical_test** - doctor orders test for patient
- **Medical_test_reports** - report generated after test
- **Bills** - bill created for appointment + test
- **Payments** - payment done against bill (upi/card)
